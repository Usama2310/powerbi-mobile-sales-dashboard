# Mobile Sales Dashboard (Power BI)

This Power BI dashboard analyzes mobile sales data to provide insights into:
- Total sales, transactions, and average sales
- Monthly sales trends
- City-wise sales performance
- Payment method distribution
- Brand and model comparison
- Customer ratings analysis

## Tools Used
- Power BI
- DAX
- Data Modeling
- Data Visualization
